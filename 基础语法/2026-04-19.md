-const修饰指针
const修饰指针有三种情况：
1.const修饰指针 --- 常量指针（像一个机械表，指针可以转，但是每个空间的数字不变）
2.const修饰常量 --- 指针常量（像一块电子表，所显示的屏幕位置不变，但是空间里的数字可以变）
3.const即修饰指针，又修饰常量
```cpp
int a = 10;
int b = 100;

const int * p1 = &a; //const右边跟的是数据类型，为常量指针，值不可改变
p1 = &b; //正确，可以修改指向

int * const p2 = &a; //const右边跟的是指针变量名，为指针常量，指向不可改变
*p2 = 30; //正确，可以修改值

const int * const p3 = &a; //称为双常量指针
```

-指针和数组
可以利用指针访问数组
```cpp
int arr[] = {1,4}
int * p = arr;
cout << arr[0] << endl;
cout << *p << endl;
p++;
cout << *p << endl;
```
输出：
```cpp
1
1
4
```
