-指针数组加函数
```cpp
void bubbleSort(int * arr,int len){
    for(int i = 0;i < len - 1;i++){
        for(int j = 0;j < len - 1 - i;j++){
            if(arr[j] < arr[j + 1]){
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

void printArray(int arr[],int len){
    for(int i = 0;i < len;i++){
        cout << arr[i] << endl;
    }
}

int main() {

    int arr[10] = { 4,3,6,9,1,2,10,8,7,5 };
    int len = sizeof(arr) / sizeof(int);

    bubbleSort(arr,len);

    printArray(arr,len);

    return 0;
}
```
求数组的长度可以用sizeof（数组名）/sizeof（数组类型）
以及void类型的函数无返回值

-结构体的定义和使用
通过结构体创建变量的方式有三种：
1.struct 结构体名 变量名    
2.struct 结构体名 变量名 = { 成员1值 ， 成员2值...}    
3.定义结构体时顺便创建变量    
```cpp
struct student
{
    string name;
    int age;
    int score;
}stu3;//第3种定义方法

int main() {

    struct student stu1;//第1种定义方法，前边结构体已经定义过了，所以这里可以省略struct（但是在C语言里不可以省略）
    stu1.name = "张";
    stu1.age = 18;
    stu1.score = 100;

    struct student stu2 = {"张",19,99};//第2种定义方法，和前边一样，可以省略struct

    return 0;
}
```
第三种定义方法还可以继续写成struct student{……}stu1，stu2，stu3； 但是并不建议，因为这里定义的变量是全局变量    
结构体的定义还可以在.h的头文件里定义，这样在主函数的前边引用头文件后就可以直接student+变量名称了
