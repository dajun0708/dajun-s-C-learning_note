-冒泡排序
```cpp
  int arr[9] = { 4,2,8,0,5,7,1,3,9 };
	for (int i = 0; i < 9 - 1; i++)
	{
		for (int j = 0; j < 9 - 1 - i; j++)
		{
			if (arr[j] > arr[j + 1])
			{
				int temp = arr[j];
				arr[j] = arr[j + 1];
				arr[j + 1] = temp;
			}
		}
	}
```

-二维数组每行求和
```cpp
int arr[3][3] = {
        {100,100,100},
        {90,50,100},
        {60,70,80}
    };
    string name[3] = {
        "三","四","五"
    };
    for(int i = 0;i < 3;i++){
        int sum =0;
        for(int j = 0;j < 3;j++){
            sum += arr[i][j];
        }
        cout << name[i] << "同学总分为：" << sum << endl;
    }
```

-函数
| 传递方式 | 函数定义 | 调用方式 | 本质 | 就像是... |
| :--- | :--- | :--- | :--- | :--- |
| **值传递** | `void swap(int n1, int n2)` | `swap(a, b)` | 拷贝副本 | 修改复印件 |
| **地址传递** | `void swap(int *n1, int *n2)` | `swap(&a, &b)` | 操控地址 | 顺着地址上门拆家 |
| **引用传递** | `void swap(int &n1, int &n2)` | `swap(a, b)` | 绑定别名 | 给本人起个外号 |

值传递不改变原值，很安全；引用传递高效修改，很推荐
